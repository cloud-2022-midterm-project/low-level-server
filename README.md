# Low-level Overengineered Server

This is a low-level, destined to fail, overengineered,  server for cloud computing midterm.

## How to run

### INSTALL RUST

```bash
https://www.rust-lang.org/tools/install
```

### .env

Create a `.env` file and fill it (see .env.example).

### Run

Debug mode:

```bash
cargo r
```

Release mode:

```bash
cargo r -r
```
